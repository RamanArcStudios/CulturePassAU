Hello! I'd love to help you modernize the user interface and experience of your app. You have a fantastic foundation here with React Native, Reanimated, and your custom design system.

To give it that premium, modern "Apple-style" feel, we can introduce a few key UX/UI upgrades:

1. **Glassmorphism (Blur Effects):** Using `expo-blur` on card overlays instead of flat dark gradients makes the UI feel native and deep.
2. **Snappy Carousels:** Adding `snapToInterval` to your horizontal `ScrollView`s makes swiping feel precise and buttery smooth.
3. **Elevated Interactions:** Applying your custom shadows from `colors.ts` and refining the "Quick Actions" grid to look like floating, tactile buttons.
4. **Premium Whitespace & Typography:** Tweaking the line heights, border radii, and padding to give the content room to breathe.

Here is how we can upgrade your `EventCard` and the main `Home` screen.

### 1. Modernizing `EventCard.tsx` (Adding Glassmorphism & Shadows)

Let's replace the basic black gradient with a sleek `BlurView` overlay for the text area. This makes the text highly readable while letting the vibrant event image shine through.

```tsx
// components/Discover/EventCard.tsx
import { View, Text, Pressable, StyleSheet, Image, Platform } from 'react-native';

interface EventCardProps {
  event: {
    id: string;
    title: string;
    date: string;
    venue?: string;
    city?: string;
    imageUrl?: string;
    communityTag?: string;
    attending?: number;
    priceLabel?: string;
    isFeatured?: boolean;
  };
  highlight?: boolean;
  index?: number;
}

function formatDate(dateStr: string): string {
  const parts = dateStr.split('-');
  if (parts.length !== 3) return dateStr;
  const day = parseInt(parts[2], 10);
  const monthIndex = parseInt(parts[1], 10) - 1;
  const months = ['Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun', 'Jul', 'Aug', 'Sep', 'Oct', 'Nov', 'Dec'];
  return `${day} ${months[monthIndex] || ''}`;
}

export default function EventCard({ event, highlight, index = 0 }: EventCardProps) {
  return (
    <Animated.View entering={FadeInDown.delay((index || 0) * 80 + 100).duration(500)}>
      <Pressable
        style={({ pressed }) => [
          styles.card,
          highlight && styles.highlight,
          pressed && { opacity: 0.9, transform: [{ scale: 0.98 }] }, // Tactile feedback
          Platform.OS === 'web' && { cursor: 'pointer' as any },
          Colors.shadows.medium, // Utilizing your design system shadows
        ]}
        onPress={() => router.push({ pathname: '/event/[id]', params: { id: event.id } })}
      >
        <Image
          source={{ uri: event.imageUrl }}
          style={StyleSheet.absoluteFillObject}
          resizeMode="cover"
        />
        
        {/* Modern Glassmorphism Overlay */}
        <BlurView intensity={80} tint="dark" style={styles.blurContainer}>
          <Text style={[styles.date, highlight && styles.dateHighlight]}>
            {formatDate(event.date)}
          </Text>
          <Text style={[styles.title, highlight && styles.titleHighlight]} numberOfLines={2}>
            {event.title}
          </Text>
          
          <View style={styles.metaRow}>
            <Ionicons name="location" size={14} color="#E5E5EA" />
            <Text style={styles.location} numberOfLines={1}>
              {event.venue || event.city}
            </Text>
          </View>

          {event.communityTag ? (
            <View style={styles.culturePill}>
              <Text style={styles.culturePillText}>{event.communityTag}</Text>
            </View>
          ) : null}
        </BlurView>
      </Pressable>
    </Animated.View>
  );
}

  },
```

### Key UI/UX Concepts Applied:

* **Tactile Feedback**: Added `scale` transforms when elements like cards or buttons are `pressed`. This makes the app feel highly responsive.
* **Snapping Behavior**: Notice `snapToInterval={296}` on the horizontal scroll views. This makes it perform like a high-end carousel rather than a free-scrolling list.
* **Larger Tap Targets & Rounded Corners**: Upped the radius on icons and cards (e.g., `borderRadius: 24`). This mimics modern iOS 17/18 and Android Material You aesthetics.
* **Depth and Layering**: Utilizing `BlurView` combined with the soft custom drop shadows (`Colors.shadows.medium`) provides excellent hierarchy.
